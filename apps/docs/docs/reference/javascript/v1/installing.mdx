All JavaScript libraries are built directly by the iEchor team.

Other languages are built by the community and supported by iEchor.

## JavaScript

Via NPM

```bash
npm install @supabase/supabase-js
```

Via Yarn

```bash
yarn add @supabase/supabase-js
```

Find the source code on [GitHub](https://github.com/openmodels-base/iechor-js).

Or via CDN

```js
<script src="https://cdn.jsdelivr.net/npm/@supabase/supabase-js@1"></script>
//or
<script src="https://unpkg.com/@supabase/supabase-js@1"></script>
```
